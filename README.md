#PrimeTax-Analytics

PrimeTax Analytics is a full-stack personal finance and tax management platform designed to help users efficiently track income, expenses, budgets, and tax estimations through an intuitive dashboard.

---

## Features

-  Secure User Authentication (Login/Register with JWT)
-  Interactive Financial Dashboard
-  Income & Expense Tracking
-  Budget Planning and Monitoring
-  Tax Estimation Calculator
-  Financial Reports Generation (PDF/CSV)
-  Visual Charts & Analytics
-  Responsive Modern UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
